# AI Infrastructure

Здесь хранится всё, что связано с кодингом AI-агентов, MCP-серверов, автоматизации и команд.

## Структура

```
ai/
├── agents/        # Определения агентов и промпты
├── mcp/           # MCP-серверы: конфиги и код
├── commands/      # Slash-команды (исходники)
└── hooks/         # Скрипты автоматизации
```

## Быстрая навигация

| Задача | Куда идти |
|--------|-----------|
| Написать промпт для агента | `ai/agents/prompts/` |
| Добавить MCP-сервер | `ai/mcp/` → обновить `.mcp.json` в корне |
| Создать slash-команду | `ai/commands/` → задеплоить в `.claude/commands/` |
| Добавить хук автоматизации | `ai/hooks/` → прописать в `.claude/settings.local.json` |

## Принцип

- `ai/` — источник правды (всё редактируется здесь, всё версионируется)
- `.claude/commands/` — рабочее место (технический путь для AI-ассистента)
- `.mcp.json` в корне — активная конфигурация MCP
