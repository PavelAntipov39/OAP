# Hooks (Автоматизация)

Хуки — скрипты, которые запускаются автоматически при определённых событиях
(до/после вызова инструментов, при отправке промпта, и т.д.).

## Как подключить хук

1. Напиши скрипт в этой папке (`.sh`, `.js`, `.py`)
2. Пропиши в `.claude/settings.local.json`:

```json
{
  "hooks": {
    "PreToolUse": [{
      "matcher": "Bash",
      "hooks": [{
        "type": "command",
        "command": "bash ai/hooks/my-hook.sh"
      }]
    }]
  }
}
```

## Доступные события

| Событие | Когда срабатывает |
|---------|------------------|
| `PreToolUse` | До вызова любого инструмента |
| `PostToolUse` | После вызова инструмента |
| `UserPromptSubmit` | При отправке сообщения |
| `Stop` | При завершении ответа |
