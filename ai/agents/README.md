# Agents

Здесь хранятся определения агентов, системные промпты и шаблоны.

## Структура

```
agents/
└── prompts/       # Переиспользуемые промпты и шаблоны
```

## Соглашения

- Каждый агент — отдельный `.md` файл с описанием роли, поведения и ограничений
- Шаблоны промптов кладём в `prompts/` и переиспользуем в коде
- Операционный реестр агентов (runtime) — в `docs/agents/registry.yaml`
