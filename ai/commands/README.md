# Commands (Slash-команды)

Slash-команды — быстрые переиспользуемые инструкции, вызываемые через `/имя-команды`.

## Как это работает

1. Напиши команду в `ai/commands/my-command.md`
2. Задеплой: скопируй файл в `.claude/commands/my-command.md`
3. Вызывай командой `/my-command` в чате с ассистентом

## Структура файла команды

```markdown
# Название команды

Краткое описание что делает команда.

## Шаги

1. Шаг первый
2. Шаг второй
3. ...
```
