# ADR Directory (OAP)

Use this folder for architecture and contract decisions that affect OAP behavior.

## Naming
- `ADR-0001-short-title.md`
- `ADR-0002-short-title.md`

## Required sections
- Context
- Decision
- Alternatives considered
- Consequences
- Rollout plan

## Rule
- Any change impacting schema/API/ETL/security should update ADR first, then code.
